NSR Reader 1.5.0 - Major Release

Enhancements

  • Refresh UI (new splash screen, updated icons, Settings)
  • Add support for OS 10.3
  • Add BlackBerry® Passport device support
  • Add text encoding autodetection (Settings->Autodetect Encoding)
  • Add bright theme support (Settings->Visual Theme)
  • Add brand colors visual tune option (Settings->Brand Colors) for OS 10.3
  • Greatly improve zooming capabilities
  • Make keyboard shortcuts more convenient (without Ctrl key)
  • Add I and O keyboard shortcuts for zoom in and zoom out actions
  • Use single taps in text reflow mode for navigation
  • Add JPEG compression support for TIFF files

Bugfixes

  • Do not use Volume keys when no file opened
  • Do not use Volume keys when file has single page
  • Fix fitting to width for cropped pages in some circumstances
  • Fix page scrolling with keyboard
