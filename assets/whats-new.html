NSR Reader 1.3.0 - Major Release

Enhancements

  • New feature to autocrop blank edges (Settings->Crop Blank Edges)
  • Add accessibility support (requires OS 10.2)
  • Replace Go to dialog with slider
  • Read orientation tag from TIFF files if available
  • Improve rendering performance a bit
  • Show information toast if Shared Files permission is disabled
  • Add Z30 device support
  • Increase space between sections in Settings

Bugfixes

  • Do not show Changes on the first ever app launch
  • Put Action Bar into fullscreen mode after failed attempt to open previous file
  • Fix fit to width feature when page is rotated into landscape mode
  • Fix crash on exit
  • Fix bug when text file page was reloaded when text reflow option has been disabled in Settings
  • Fix bug when after zooming text in reflow mode can disappear
NSR Reader 1.2.0 - Major Release

Enhancements

  • Add active frame (application cover) support
  • Always save last position by default (option removed from the Settings)
  • Use sharing dialog to select apps to share through
  • Add multi-selection support (including share functionality) on Recent page
  • Add Delete action (removes files from the phone) for recent files
  • Show undo popup toasts when clearing recent files from the list
  • Show encryption icon and do not show file contents in thumbnail for password protected files on Recent page
  • Save last position for every page in the cache
  • Add keyboard shortcuts for Open and Go to actions
  • Add list of keyboard shortcuts to Help section of About page
  • Add more text encodings in Settings
  • Redesign spalsh screen
  • Update some icons
  • Overall UI cleanup according to BlackBerry® 10 UI Guidelines document

Bugfixes

  • Fix processing card resize message
  • Improve checking for outdated thumbnails on Recent page
  • Reset app on card pooling event properly
  • Improve security by handling access to files more strictly
  • Fix using Shift JIS encoding
NSR Reader 1.1.0 - Major Release

Enhancements

  • Add page rotation support (see page's context menu)
  • Integration with BlackBerry® 10 (i.e. you can open files in NSR Reader from File Manager)
  • NSR Reader can be used as a card from third-party apps
  • Improve cache behavior
  • Rework About page with new Help and Changes sections
  • Save last position in text reflow mode
  • Save page horizontal position when loading a new one
  • Make pinch-to-zoom relative to the center of the view
  • Use dark theme by default
  • Change progress reading indicator's color to orange

Bugfixes

  • Fix crash when using Recent Documents page
  • Fix crash when using large zoom values
  • Last page in TIFF was not loaded
  • Remove flickering when fitting page to width
  • Remove flickering when showing Action Bar in fullscreen mode
  • Show notification after removing all items on Recent Documents page
  • Check image size for device hardware limits
